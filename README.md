# Full Stack Blog App

Following [this]([https://](https://youtu.be/Pfy3q6PbbRA?si=37zJlGS-BmVxheMI&t=21423)) tutorial

Startup commands:
```
cd client
npm run dev
```
```
cd backend
node --env-file .env --watch index.js
ngrok http http://localhost:3000
```

Built using:
- Clerk
- ImageKit
- MongoDB
- Ngrok
